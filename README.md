An XrmToolbox plugin to help find which components Dataverse fields are used in.

<img width="1898" height="1243" alt="image" src="https://github.com/user-attachments/assets/a55a8b60-f217-4a14-9162-47fd17bb8364" />

To use the Field Reference Finder:
   1. Connect to an Environment and your tables will load into the Table dropdown
   2. Select a table where the field is that you would like to analyze
   3. The Field dropdown will populate with all the fields from the selected Table
   4. Select the field you want to analyze
   5. On the right side, check which component types you want to search (Web Resources, Processes, Cloud Flows, Canvas Apps, Forms, Views)
   6. Click Search
   7. The results will be displayed in the table at the bottom.
   8. Click Export if you would like a CSV version of the table.
